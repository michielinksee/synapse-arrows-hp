Privacy Policy

Last Updated: July 1, 2026

Synapse Arrows Pte. Ltd.（以下「当社」）は、当社が開発・運営するプロダクトおよびサービスにおけるユーザーの個人情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。

対象プロダクト・サービス

本ポリシーは以下の全プロダクトに適用されます：

  • KanseiLINK — AEOレーティングプラットフォーム（kansei-link.com）
  • ScaNavi — 日本酒インテリジェンスアプリ（scanavi.app / iOS / Android）
  • Linksee Memory — エージェント記憶MCPサーバー（OSS）
  • Bantou（番頭） — AI自動経理・税務分類エージェント
  • CardWize — AIカード管理プラットフォーム
  • ScaNavi-PC — PC向け家電レビュー・比較サービス
  • ScaNavi-Kaden — 家電インテリジェンスサービス
  • 当社が提供するコンサルティング・アドバイザリーサービス

1. 収集する情報

1.1 アカウント情報

  • メールアドレス（認証用）
  • 氏名（任意）
  • Google / Apple アカウント情報（ソーシャルログイン利用時）
  • ユーザー設定・プリファレンス

1.2 利用データ

  • 各プロダクトの機能利用状況
  • 検索クエリ・スキャン履歴（ScaNavi）
  • AEOスコア参照履歴（KanseiLINK）
  • エラーログ・クラッシュレポート
  • デバイス情報（OS、ブラウザ、端末種別）

1.3 プロダクト固有のデータ

ScaNavi（モバイル・Web）:

  • カメラで撮影した日本酒ラベル画像（AI解析後、サーバーに恒久保存しません）
  • お気に入り・テイスティングノート
  • 位置情報（ユーザーが許可した場合のみ、近隣店舗検索用）

KanseiLINK:

  • MCPサーバー経由のAPI利用ログ
  • サービス検索・レポート送信データ

Bantou（番頭）:

  • freee等の会計SaaS連携時に取得する取引データ（API経由）
  • 分類結果・修正履歴
  • 取引データはfreeeのサーバーに保存され、当社サーバーには恒久保存しません

Linksee Memory:

  • Linksee Memoryは完全ローカル動作のOSSです。ユーザーのデータは当社に送信されません
  • GitHubリポジトリの利用統計（npmダウンロード数等）のみ匿名で収集

2. 情報の利用目的

  • プロダクトの機能提供・改善
  • AIによるレコメンデーション・分析の実行
  • ユーザーサポートへの対応
  • 匿名化された統計分析によるサービス改善
  • セキュリティの確保・不正利用の防止
  • 法令に基づく対応

3. AI サービスの利用

当社のプロダクトは、以下のAIサービスプロバイダーを利用してコンテンツ生成・分析を行います：

  • Anthropic（Claude）— テキスト生成、分類、分析
  • Google（Gemini）— 画像認識、テキスト生成

AIプロバイダーに送信されるデータは、各プロバイダーのプライバシーポリシーに従い処理されます。送信データがAIモデルのトレーニングに使用されることはありません（各プロバイダーのAPI利用規約に基づく）。

4. Google API Services User Data Policy

当社プロダクトのうち、Google APIを利用するもの（ScaNavi等）は、Google API Services User Data Policy（Limited Use requirements を含む）に準拠しています。

  • Googleユーザーデータは本ポリシーに記載された機能の提供・改善のみに使用します
  • サービス提供に必要な場合を除き、第三者に転送しません
  • 広告目的には使用しません
  • ユーザーの承認、セキュリティ上の必要、または法的要件がない限り、人がデータを閲覧することはありません

5. データの保存・セキュリティ

5.1 保存場所

  • クラウドストレージ: Supabase（米国）、Vercel（グローバルCDN）
  • 一時処理: AI解析データは処理後に恒久保存しません
  • ローカル処理: Linksee Memory はユーザー端末内のSQLiteに保存（サーバー送信なし）

5.2 セキュリティ対策

  • 全通信のHTTPS/TLS暗号化
  • OAuth 2.0 による認証
  • 定期的なセキュリティ監査
  • アクセス制御とモニタリング

6. 第三者への提供

当社はユーザーの個人データを販売しません。以下の場合に限り、限定的にデータを共有することがあります：

  • AIサービスプロバイダー（上記3項に記載）— 機能提供に必要な範囲のみ
  • 法的要件に基づく場合
  • ユーザーの明示的な同意がある場合

7. データの保持期間

  • アカウントデータ: アカウントが有効な間保持
  • 一時処理データ（画像・メール等）: 処理完了後に削除
  • 利用統計: 匿名化の上、サービス改善のために保持
  • ユーザーはいつでもデータの削除を請求できます

8. ユーザーの権利

ユーザーは以下の権利を有します：

  • アクセス権: 保有データのコピーを請求する権利
  • 訂正権: 不正確な情報を更新する権利
  • 削除権: データの削除を請求する権利
  • アクセス取り消し: GoogleアカウントやSNS連携の解除
  • データポータビリティ: データのエクスポートを請求する権利

9. 児童のプライバシー

当社のプロダクトは16歳未満の児童を対象としていません。ScaNaviは酒類に関するサービスのため、日本国内においては20歳以上の利用者を対象としています。児童から意図的に個人情報を収集することはありません。

10. Cookie・トラッキング

当社のWebサイトおよびWebアプリケーションでは、サービス提供に必要なCookieを使用します。アクセス解析にはプライバシーに配慮したツールを使用し、ユーザーの個人を特定するトラッキングは行いません。

11. ポリシーの変更

本ポリシーは必要に応じて更新されます。重要な変更がある場合は、プロダクト内通知またはメールにてお知らせします。変更後のサービスの継続利用をもって、更新されたポリシーに同意したものとみなします。
